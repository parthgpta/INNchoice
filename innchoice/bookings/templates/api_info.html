Using API's
Introduction

With Innchoice API you can get access to some of our data in machine-readable JSON format.

To access the data you just send a HTTP-request to address" /api/{methodname} " with method-specific parameters. Each method description has an example URL.


Method Names

  • accounts/listusers

    Returns a list of all registered Users , informations and their User Types.
    Returned Objects Example- "
    [
    {
    "user": 1,
    "name": "user1",
    "gender": "Male",
    "dob": "1985-10-12",
    "phone": 9559889632,
    "user_type": "Room Manager"
    },
    { "user": 2,
    "name": "user2",
    "gender": "Male",
    "dob": "1958-10-02",
    "phone": 9988995566,
    "user_type": "Customer"
    }
    ]"
    URL - " /api/accounts/listusers "


  • accounts/listusers/create/signup

    User can signup themselves , they need to send a Username and Password .
    Example - " { "username": "newuser", "password: "******" }"
    URL - " /api/accounts/listusers/create/signup "


  • accounts/listusers/create/profile

    (Users have to signup on above link First)
    User can create Profile for themselves on InnChoice , they need to send a user(as created in the previous link) ,name, gender ,dob , phone and Usertype (Customer or Room Manager) .
    Example - " [
    {
    "user": newuser,
    "name": "newuser",
    "gender": "Male",
    "dob": "1985-10-12",
    "phone": 9559889632,
    "user_type": "Room Manager"
    },
    ]
    "
    URL - " /api/accounts/listusers/create/profile"


  • bookings/rooms/lists

    Returns a list of all available rooms and their details .
    Returned Objects Example- "
    [
    {
    "number": 28,
    "day": 21,
    "month": 3,
    "year": 2020,
    "room_type": "Suite",
    "starttime": 12
    "endtime": 17
    },
    {
    "number": 16,
    "day": 22,
    "month": 3,
    "year": 2020,
    "room_type": "Deluxe",
    "starttime": 7
    "endtime": 11
    }
    ]"
    URL - " /api/bookings/rooms/lists "


  • bookings/timeslots/list

    Returns a list of all available timeslots.
    Returned Objects Example - " [
    {
    "starttime": 12,
    "endtime": 17
    },
    {
    "starttime": 7,
    "endtime": 11
    }
    ]"
    URL - " /api/bookings/timeslots/list "


  • booked/list/<name>

    Returns a list of all bookings of a Specific customer . User have to send a string name
    Returned Objects Example - " [
    {
    "customer": <name>,
    "manager": "newmanager",
    "date": 21,
    "month": 3,
    "year": 2020,
    "room_t": "Suite",
    "starttime": 7,
    "endtime": 11
    },
    {
    "customer": <name>,
    "manager": "newmanager",
    "date": 22,
    "month": 3,
    "year": 2020,
    "room_t": "Deluxe",
    "starttime": 12,
    "endtime": 17
    },
    ]
    ""
    URL - " /api/bookings/booked/list/<name> "
